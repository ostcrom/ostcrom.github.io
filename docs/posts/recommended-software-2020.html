Recommended Software (2020)

Posted Sun 11 October 2020   | Category : Recommendations

A few applications that I find worth using in 2020.

tmux - *Nix terminal multiplexer, allows you to easily multitask in a terminal by providing resizable terminal 'windows'.

ohmyzsh - A quote from the project's README describes this terminal enhancer accurately: "Oh My Zsh will not make you a 10x developer...but you may feel like one."

Portainer - A super easy to use web interface for Docker and recently also Kubernetes.

macpass - Use to generate, store and retrieve unique credentials on macOS.
